# SolarTech: Solar Energy Business Website
A fully responsive multi-page business website built for a solar energy company as a frontend portfolio project.
Live Demo: https://zmaham774.github.io/solar-company-website/

# Pages
- Home: Hero slider, services overview, testimonials
- About: Company intro, mission & vision, team section
- Services: Full solar services with descriptions
- Projects: Auto-playing image sliders by category
- Contact: Split screen form, Google Map, info cards

# Built With
- HTML5
- CSS3
- Tailwind CSS
- JavaScript 
- AOS Scroll Animations
- Lucide Icons

# Features
- Fully responsive on mobile, and desktop
- Auto-playing hero image slider with dot navigation
- Project sliders with pause on hover and progress bar
- Scroll-triggered counter animations
- Navbar hides on scroll down, shows on scroll up
- Contact form with loading animation and success message
- Mobile hamburger menu
- AOS scroll animations throughout
- Google Maps embed on contact page

# Note
This is a frontend project. The contact form is for UI demonstration only.
